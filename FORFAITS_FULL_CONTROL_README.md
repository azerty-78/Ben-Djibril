## 1. Introduction

Ce document présente de manière détaillée tous les forfaits Full Control proposés par Ben Djibril. Il sert de référence complète pour comprendre les modalités, les fonctionnalités incluses, les conditions d'utilisation et les engagements de chaque forfait.

**Objectif du document :** Fournir une documentation exhaustive pouvant servir de base pour l'élaboration de contrats commerciaux et de spécifications techniques.

**Principe fondamental :** Dans le modèle Full Control, le client devient propriétaire à 100% du code source et de toute la propriété intellectuelle une fois les jalons et paiements prévus atteints.

## 2. Vue d'ensemble du Modèle Full Control

### 2.1 Définition

Le modèle Full Control (Contrôle Total) consiste en un **achat définitif** d'une application web personnalisée. Dans ce modèle :

- **Le client** paie une fois et possède l'application à vie
- **Le développeur (Ben Djibril)** développe l'application selon les spécifications
- **La propriété intellectuelle et le code source** sont transférés au client à 100%
- **Le client** a une autonomie complète : peut héberger où il veut, modifier le code, même revendre

### 2.2 Caractéristiques Communes à Tous les Forfaits Full Control

Tous les forfaits Full Control incluent les éléments suivants :

✅ **Code source** - 100% propriété du client, transfert complet à la livraison  
✅ **API REST (Backend)** - Architecture backend complète et documentée  
✅ **Front-end** - Interface utilisateur complète et responsive  
✅ **Documentation** - Documentation technique et utilisateur (40-200 pages selon forfait)  
✅ **Serveur Virtuel Privé (VPS)** - Personnalisable selon vos besoins (année 1 incluse)  
✅ **Certificat SSL** - Sécurité HTTPS incluse  
✅ **Nom de domaine personnalisé** - Inclus dans le forfait  
✅ **Période de garantie** - 1 à 6 mois selon forfait  
✅ **Formation** - Sessions de formation incluses (2-4 sessions selon forfait)  
✅ **Support post-livraison** - Support technique inclus (1-6 mois selon forfait)

### 2.3 Délais de Livraison

Les délais de livraison varient selon le forfait :

- **Ultra Speed** : 65 jours (60 jours de développement + 7 jours de buffer)
- **Speed** : 110 jours (90 jours de développement + 14 jours de buffer)
- **Normal** : 180 jours

### 2.4 Public Cible

Le modèle Full Control est idéal pour :
- Les entreprises établies nécessitant une autonomie totale
- Les projets stratégiques où la propriété intellectuelle est critique
- Les entreprises avec équipe technique disponible
- Les projets nécessitant des personnalisations importantes
- Les projets à long terme nécessitant une architecture scalable

## 7. Conditions Générales

### 7.1 Propriété Intellectuelle

**Dans le modèle Full Control :**

- ✅ **Code source et propriété intellectuelle** : 100% propriété du client après paiement complet
- ✅ **Transfert complet** : Tous les droits d'auteur sont cédés au client
- ✅ **Autonomie totale** : Le client peut modifier, distribuer, revendre ou créer des produits dérivés
- ✅ **Livrables** : Code source complet (Git repository), API REST backend, frontend, schéma base de données, documentation, tous les assets associés

### 7.2 Arrêt de la Collaboration

Si le client arrête la collaboration :

- ✅ Le client conserve **tout le code livré** et la documentation selon le contrat signé
- ✅ Le client peut continuer avec son équipe interne ou un autre prestataire
- ✅ Code source 100% propriété du client
- ✅ Le client peut migrer ses données vers une autre solution
- ✅ Support migration disponible (optionnel, facturé séparément)

### 7.3 Utilisation des Données

- Les données du client sont hébergées sur le VPS fourni (année 1)
- Les sauvegardes sont effectuées selon le niveau du forfait
- Le client a un contrôle total sur ses données
- Après l'année 1, le client gère son hébergement ou souscrit à un abonnement maintenance

## 9. Stack Technique et Infrastructure

### 9.1 Technologies Utilisées

Tous les forfaits Full Control utilisent des technologies modernes, robustes et scalables :

#### Backend
- **Spring Boot** (Kotlin/Java) - Framework backend robuste
- **RESTful API** avec authentification JWT
- **MongoDB** - Base de données NoSQL pour gestion flexible des données

#### Frontend
- **React + Vite** - Framework frontend moderne et rapide
- **Tailwind CSS** - Framework CSS utility-first
- **TypeScript** - Typage statique pour code robuste

#### Sécurité
- **JWT tokens** - Authentification sécurisée
- **HTTPS/SSL** - Connexion chiffrée
- **Validation inputs** - Prévention injection SQL
- **Protection XSS/CSRF** - Protection contre attaques web
- **Firewall** - Protection serveur

#### Infrastructure
- **VPS (Ubuntu)** - Serveur virtuel privé
- **Nginx** - Serveur web et reverse proxy
- **PM2** - Gestionnaire de processus Node.js
- **Docker** - Conteneurisation (forfait Normal)
- **CI/CD** - Pipelines automatisés (forfait Normal)

#### Compléments
- **Redis** - Cache (Speed & Normal)
- **Docker & CI/CD** - Automatisation (Normal)

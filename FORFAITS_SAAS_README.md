## 2. Vue d'ensemble du Modèle SaaS

### 2.1 Définition

Le modèle SaaS (Software as a Service) proposé consiste en une **licence mensuelle** d'utilisation d'une application web personnalisée. Dans ce modèle :

- **Le client** paie une licence mensuelle pour utiliser l'application
- **Le développeur (Ben Djibril)** gère les serveurs, la maintenance et les mises à jour
- **La propriété intellectuelle et le code source** restent la propriété du développeur
- **L'accès à l'application** est garanti 24h/24 et 7j/7

### 2.2 Caractéristiques Communes à Tous les Forfaits SaaS

Tous les forfaits SaaS incluent les éléments suivants :

✅ **Hébergement sécurisé** - L'application est hébergée sur des serveurs sécurisés  
✅ **Maintenance & mises à jour de sécurité** - Maintenance régulière et mises à jour de sécurité  
✅ **Accès à l'application 24h/24 et 7j/7** - Disponibilité continue de l'application  
✅ **Certificat SSL (HTTPS)** - Connexion sécurisée et chiffrée  
✅ **Sauvegardes automatisées** - Sauvegardes régulières des données  
✅ **Support technique** - Assistance technique selon le niveau du forfait  
✅ **Option de facturation annuelle avec économies** - Réduction de 16% avec paiement annuel

### 2.3 Délai de Livraison

**Tous les forfaits SaaS sont livrés avec un délai maximum de 45 jours** à compter de la validation du cahier des charges et du paiement de l'acompte.

### 2.4 Public Cible

Le modèle SaaS est idéal pour :
- Les entrepreneurs et entreprises qui veulent lancer rapidement
- Ceux qui souhaitent un coût mensuel maîtrisé
- Les clients qui ne souhaitent pas gérer les serveurs, le déploiement ou la maintenance technique

## 3. Forfait Good Deal

### 3.1 Description Générale

**Forfait accessible pour lancer votre plateforme personnalisée à moindre coût.**

Le forfait Good Deal est conçu pour les entrepreneurs solos, les petites activités locales et les premiers sites web. Il offre les fonctionnalités de base nécessaires pour démarrer une activité en ligne.

### 3.2 Prix et Facturation

#### Prix Mensuel
- **Prix mensuel :** 15 500 F CFA (HT) / mois
- **Prix annuel :** 156 000 F CFA (HT) / an
- **Économie annuelle :** 30 000 F CFA par rapport au paiement mensuel

#### Calcul des Économies
- 12 mois × 15 500 F = 186 000 F CFA
- Paiement annuel : 156 000 F CFA
- **Économie :** 30 000 F CFA (16%)

### 3.3 Fonctionnalités Incluses

#### Infrastructure
- **Stockage :** 10 Go de base + facturation supplémentaire au besoin (pay-as-you-grow)
- **Nom de domaine professionnel :** www.mon_domaine.com
- **Email professionnel :** …@mon_entreprise.com
- **Hébergement :** Cloud sécurisé mutualisé
- **Sauvegardes :** Hebdomadaires (pas de sauvegarde cloud incluse)

#### Fonctionnalités de Base
1. **MVP : Minimum Viable Product** - Version minimale viable avec fonctionnalités essentielles
2. **2 interfaces de gestion :**
   - Interface Administrateur
   - Interface Utilisateur simple (client, etc.)
3. **CRUD fonctionnel :** 4 fonctionnalités de base (Create, Read, Update, Delete)
4. **Recherche :** 1 fonctionnalité de recherche simple
5. **SEO :** SEO et référencement élémentaire
6. **Sécurité :** Sécurité basique
7. **Publicité :** Annonces publicitaires incluses

### 3.4 Cadence d'Évolution

- **Fréquence :** 1 nouvelle fonctionnalité développée par an (1 fois tous les 12 mois)
- **Badge cadence :** 1/an
- **Période de validité :** Les nouvelles fonctionnalités incluses sont valables pendant **6 ans** après la souscription au forfait

**Note importante :** Après cette période de 6 ans, toute nouvelle fonctionnalité nécessitera le paiement de son développement. Il est également possible de payer le développement d'une fonctionnalité à tout moment si le nombre de fonctionnalités est épuisé avant les 6 ans.

### 3.5 Support Technique

- **Niveau :** Support email
- **SLA (Service Level Agreement) :** Réponse sous 48h ouvrées
- **Canaux :** Email uniquement

### 3.6 Idéal Pour

- Entrepreneurs solos
- Petites activités locales
- Premiers sites web
- Projets nécessitant un budget limité

Si le client arrête l'abonnement :

## 9. Politique d'Évolution

### 9.1 Fonctionnalités Incluses

Chaque forfait inclut un certain nombre de **nouvelles fonctionnalités** développées selon une **cadence définie** :

- **Good Deal :** 1 nouvelle fonctionnalité par an (valable 6 ans)
- **Pro :** 1 nouvelle fonctionnalité par semestre (valable 2 ans)
- **Ultra :** 1 nouvelle fonctionnalité par trimestre (valable 1 an)

### 9.2 Période de Validité

Les nouvelles fonctionnalités incluses sont **valables pendant une période déterminée** après la souscription :

| Forfait | Période de Validité |
|---------|---------------------|
| Good Deal | 6 ans |
| Pro | 2 ans |
| Ultra | 1 an |

**Important :** Après cette période, toute nouvelle fonctionnalité nécessitera le paiement de son développement.

### 9.3 Développement Supplémentaire

Il est possible de **payer le développement d'une fonctionnalité** à tout moment si :

- Le nombre de fonctionnalités incluses est épuisé
- Le client souhaite une fonctionnalité supplémentaire
- La période de validité est dépassée

### 9.4 Définition des Termes

- **MVP (Minimum Viable Product)** : Version minimale viable avec fonctionnalités essentielles pour lancer rapidement
- **MVPA (Minimum Vital Product Advanced)** : Version améliorée avec plus de fonctionnalités et capacités
- **CRUD** : Create, Read, Update, Delete - Les 4 opérations de base pour gérer les données
